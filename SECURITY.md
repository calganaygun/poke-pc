# Security Policy

## Supported Versions

This project is currently in early release.

| Version | Supported |
| --- | --- |
| 0.1.x | Best effort |
| < 0.1.0 | No |

## Reporting a Vulnerability

Please open a private GitHub security advisory for this repository.

If private advisory is not available, open a regular issue with minimal details and request a private channel. Avoid publishing exploit details before maintainers can assess and patch.

## Response Expectations

This is a personal/open project and does not provide SLA guarantees. Reports are handled on a best-effort basis.
