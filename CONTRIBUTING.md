# Contributing

Thanks for your interest in contributing to Poke PC.

## Development Setup

1. Install Node.js 22+.
2. Install dependencies:
   - `npm install`
3. Run checks:
   - `npm run check`
   - `npm run build`

## Local Run

- Development: `npm run dev`
- Docker: `docker compose up --build`

## Pull Request Guidelines

1. Keep changes focused and small.
2. Add or update docs when behavior changes.
3. Ensure `npm run check` and `npm run build` pass.
4. Include clear testing notes in PR description.

## Commit Guidance

Use clear commit messages describing what changed and why.

## Reporting Bugs

Open a GitHub issue with:
- Steps to reproduce
- Expected result
- Actual result
- Logs or screenshots if available
